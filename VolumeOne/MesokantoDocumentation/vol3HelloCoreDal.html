Kapitel 3 . Projekt HelloCoreDal . EntityFramework - Data Access Layer

Archtiktur

Das Projekt HelloCoreDal fasst drei Elemente einer gegliederten Architektur zusammen:

Domain Model
bildet das Datenmodell ab, hier mit den Mitteln des EF umgesetzt
Data Access Layer
realisiert den Datenbankzugriff, ebenfalls mittels EF
Repository
stellt eine abstrakte Schnittstelle zur die Datenbank zur Verfügung sowie eine konkrete Implementierung für den Zugriff über EF

Da hier nur eine Datenbánk - MS SslServer - und ein einfaches Datenmodell verwendet werden, ist diese Zusammenfassung in einem Projekt legitim.

Benötigte Packages

Der CommandLineParser benötigt ein einziges NuGet Package:

  • CommandLineParser (hier verwendet: V2.7.82)
